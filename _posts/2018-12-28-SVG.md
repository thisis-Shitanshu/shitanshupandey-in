# #Intro
- SVG is an image format.
    - Vector based.
        - Vectors are all about polygons and shapes.
        - Vectors graphics are not for photographs.
            - Instead for illustrations, logos, icons.
        - Scale infinitely.
        - *JPG, PNG are Raster based.*
- Free Vector software:
    - [INKSCAPE](https://inkscape.org/).
    - [svg-edit](https://svg-edit.github.io/svgedit/releases/latest/editor/svg-editor.html).
    - [SVG OMG: Optimization](https://jakearchibald.github.io/svgomg/).
- SVG by default, grown and shrink with the size.
    - You can of course, specify fixed width.

## #With CSS
- Manipulate Vector Graphics with CSS for a website:
    - Inline: Include SVG directly inside html page, instead of loading it in `img` tag.
    - Give it a class/id and apply style to it.

    ```html
    <body>
        <svg width="640" height="480" xmlns="http://www.w3.org/2000/svg" xmlns:svg="http://www.w3.org/2000/svg">
            <!-- Created with SVG-edit - https://github.com/SVG-Edit/svgedit-->
            <g class="layer">
                <title>Layer 1</title>
                <rect class="my-shape" fill="#ff7f00" height="50" id="svg_2" stroke="#000000" stroke-width="5" width="134" x="65" y="107"/>
            </g>
        </svg>

        <style>
            .my-shape {
                background-color: red;
            }
        </style>
    </body>
    ```

    - With `symbol` tag:

    ```html
    <body>
        <svg><use xlink:href="#example" href="#example"></use></svg>

        <svg width="640" height="480" xmlns="http://www.w3.org/2000/svg" xmlns:svg="http://www.w3.org/2000/svg">
            <symbol id="example" class="example">
                <!-- Created with SVG-edit - https://github.com/SVG-Edit/svgedit-->
                <g class="layer">
                    <title>Layer 1</title>
                    <rect class="my-shape" fill="#ff7f00" height="50" id="svg_2" stroke="#000000" stroke-width="5" width="134" x="65" y="107"/>
                </g>
            </symbol>
        </svg>

        <style>
            .my-shape {
                background-color: red;
            }
        </style>
    </body>
    ```

# #SVG `viewBox`
- viewport (width, height) is different from viewBox.
    - What size someone is looking at the svg canvas is viewport.
- `viewBox` on svg tag: It shows what shows up inside the viewport area.

```xml
<!--viewBox="StratingCoordinateX StratingCoordinateY Width Height"-->
<svg viewBox="0 0 250 250" width="250" height="250">
</svg>
```

# #SVG Animation
- While applying scaling transform to SVG, note:
    - For HTML element transform scale works towards its own center by default.
    - For SVG transform-origin is TOP LEFT corner of the illustration.
        - You need to set `transform-origin`

    ```css
    .svg .element {
        transform: scale(.5) rotate(180deg);
        transform-origin: 340px 390px;
    }
    ```

- You can group multiple vectors with `g` tag:

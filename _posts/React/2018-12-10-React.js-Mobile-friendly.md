Notes on Implemting Style Components for Mobile Responsive View.

1. Implement Global Styles with `styled-components` using `createGlobalStyle`:

```js
// global.styles.js
import { createGlobalStyle } from 'styled-components';

export const GlobalStyle = createGlobalStyle`
    body {
        font-family: 'Open Sans Condensed';
        padding: 20px 40px;

        @media screen and (max-width: 800px) {
            padding: 10px;
        }
    }

    a {
        text-decoration: none;
        color: black;
    }

    * {
        box-sizing: border-box;
    }
`;
```

1. Include it in `App.js` and implement it above `Header` Component.
